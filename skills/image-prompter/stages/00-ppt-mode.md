# PPT模式：快速入口

## 触发条件

用户输入满足以下任一条件：
- 上传 `.md` 文件，内容包含 `第X页`、`## 第`、`可视化类型：`、`ASCII` 等标记
- 用户明确说："这是PPT大纲"、"PPT配图"、"按PPT模式"
- 文件中有 `.title-card`、`.two-col`、`.three-col`、`.grid-card` 等布局标记

## 模式识别逻辑

检测到以下信号时，自动进入PPT模式：
```
- 文件包含 "第1页"、"第2页" 等分页标记
- 包含 "可视化类型：" 字段
- 包含 ASCII 框图
- 包含 "逐字稿" 或 "内容要点"
```

## PPT模式流程（5阶段压缩为3阶段）

| 阶段 | 名称 | 目标 | 与传统模式的区别 |
|---|---|---|---|
| P1 | 解析确认 | 解析大纲文件，提取每页布局+文案，用户确认页数 | 跳过"拆内容"，直接读取已有结构 |
| P2 | 风格配色 | 选择风格（奶油纸/扁平化等）+ 确认配色方案 | 文案已固定，只调视觉 |
| P3 | 批量生成 | 为每页生成完整提示词（风格+布局+文案） | 一次性输出全部提示词 |

## PPT模式输入格式

用户提供的文件应包含：
```markdown
## 第1页：封面
**可视化类型：** `.title-card`
```
[ASCII 布局]
```
**内容要点：**
- 主标题：xxx
- 副标题：xxx
```

## PPT模式输出格式

每页生成一个提示词块：
```markdown
### 第X页：[页面名称]

**布局类型：** `two-col` | `three-col` | `title-card` | ...
**风格：** cream-paper
**配色：** 米色底 + 青 + 珊瑚

**提示词：**
```
奶油纸纹理，水彩手绘风格。严格[布局描述]。
[提取的标题]
[提取的内容点]
[视觉元素描述]
```
```

## 与传统模式的差异

| 维度 | 传统模式 | PPT模式 |
|---|---|---|
| 输入 | 文章/段落 | PPT大纲文件 |
| 文案 | 需要提炼 | 已固定，直接使用 |
| 布局 | 需要规划 | 已存在，解析复用 |
| 确认点 | 4个问题 + 风格 | 确认解析结果 + 风格配色 |
| 输出速度 | 5阶段 | 3阶段，更快 |

## 配色方案模板

PPT模式下，提供3种预设配色供选择：

**A. 奶油商务**（默认）
- 底色：米色/奶油纸
- 强调色：青 + 珊瑚
- 适合：商务演示、技术分享

**B. 清新活力**
- 底色：浅灰白
- 强调色：绿 + 橙
- 适合：产品发布、年轻受众

**C. 专业深蓝**
- 底色：白/浅蓝
- 强调色：深蓝 + 金
- 适合：金融、保险、B2B
