# 极简手绘笔记风格块（风格基准：必须以此为唯一基础）

- 画幅：3:4 竖版（封面场景需指定 21:9 超宽横版）
- 背景：纯白色，干净无纹理
- 线条：黑色粗线条（2-3px）、干净但有手绘感、略带不规则
- 阴影填充：使用斜线阴影（hatching/cross-hatching）表现立体感和层次
- 色彩点缀：
  - 黄色高亮块：不规则手绘边缘的黄色背景块，用于强调文字
  - 青色/蓝绿色（#4ECDC4）：用于圆形图标背景、边框、对话框
  - 红色：用于下划线、删除线、警示标记
- 图标风格：等距3D视角或带阴影线的扁平图标，精致有细节
- 字体：粗体黑字，中英混排，标题大字，说明小字
- 装饰元素：
  - 通用：手绘粗箭头（有弯曲感）、对话气泡、对话框、星星、波浪线
  - 封面专用装饰：
    - 四角手绘边框：粗黑线（3-4px）勾勒的四角装饰，略带不规则手绘感
    - 右下角印章区域：红色圆形或方形印章图案，内含简约文字或符号
    - 顶部/底部笔刷纹理：不规则手绘线条组成的装饰条，增加层次感
    - 斜线阴影块：作为背景层次的几何装饰块，使用斜线填充
    - 中式云纹点缀：简约的云纹或回纹装饰，用于角落或边缘
- 布局：灵活多样，支持循环、对比、分层、中心辐射等结构
- 禁止：无渐变、无3D渲染、无照片写实、无阴影（除阴影线外）
- 可读性：中文清晰可读，手绘感与干净感平衡
- 负面约束：no watermark, no logo, no random letters, no gibberish text, no photorealistic, no gradients, no clean vector look
