# 运行默认与图片比例参考

## 默认值
- 平台：公众号 + 小红书
- 字数：公众号约 1500 字；小红书约 800 字
- 图片清晰度：2K
- 模型：doubao-seedream-4-5-251128

## Ark 图片比例参数
- 官方文档：https://www.volcengine.com/docs/82379/1541523?lang=zh
- 使用 Ark `images.generate` 的 `size` 参数指定分辨率与比例。
- 公众号：16:9 横屏 = `2560x1440`
- 小红书：3:4 竖屏 = `1728x2304`
- 若 `size` 参数报错：先用 `2K` 并在 prompt 写明比例，必要时后处理裁切。
