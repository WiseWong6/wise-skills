## Skills 目录

### 🎯 omega-prompt-forge

**作用**：从零创建高质量的 AI 提示词

当你需要：
- 设计全新的 Prompt 框架
- 创建任务专用的系统提示词
- 为特定场景定制 AI 行为

**工作流程**：
1. 需求分析 - 收集任务目标、受众、场景、约束
2. 核心字段提取 - 填充角色、目标、背景、输出格式等关键槽位
3. 智能框架推荐 - 根据复杂度自动匹配最佳框架（RTF/RACE/CRISPE 等）
4. 方案确认 - 输出结构化提示词方案供确认
5. 保存输出 - 生成并保存 Markdown 格式提示词

**特点**：
- 自动判断任务复杂度（简单/中等/复杂）
- 支持 10+ 种提示词框架
- 融合 OmegaPromptForge 增强层
- 规范化保存到指定目录

### ✏️ prompt-version-editor

**作用**：严格变更控制下的提示词版本管理

当你需要：
- 局部优化现有提示词
- 修复提示词失败问题
- 小修小补而非重写
- 维护提示词版本历史

**工作流程**：
1. 变更范围确认 - 明确编辑范围，范围外内容严格保持不变
2. 版本定位 - 自动读取最新版本或创建 v1.0
3. 结构化诊断 - 检查 MECE、歧义、逻辑冲突、安全漏洞
4. 精准修改 - 仅修改批准范围内的内容
5. 版本保存 - 生成新版本并保存到 `/Users/wisewong/Documents/Developer/prompts/`

**与大改重写的区别**：
| 场景 | 使用工具 |
|-----|---------|
| 局部修订、补丁修复 | prompt-version-editor |
| 大改重写、结构重构 | omega-prompt-forge |

### 📊 ppt-speech-creator

**作用**：自动生成 PPT 结构和配套演讲逐字稿

当你需要：
- 准备年终总结/述职报告
- 项目复盘演示
- 产品发布/路演
- 任何结构化演讲内容

**工作流程**：
1. 需求分析 - 识别 PPT 类型、素材、约束条件
2. 结构规划 - 根据类型选择专用模板结构
3. 时长分配 - 自动计算或按指定总时长分配每页演讲时间
4. 内容生成 - 生成标题、要点、数据、表格等结构化内容
5. 逐字稿生成 - 按页生成配套演讲词，字数与时长精准匹配
6. 合并输出 - 统一 Markdown 格式，可直接用于制作 PPT

**支持场景**：
- 📅 年终总结：回顾 → 成果 → 问题 → 成长 → 规划
- 📁 项目复盘：背景 → 目标 → 过程 → 结果 → 经验 → 后续
- 🚀 产品发布：痛点 → 方案 → 产品 → 优势 → 市场 → 愿景
- 👔 述职报告：职责 → 业绩 → 亮点 → 不足 → 规划

**智能时长计算**：
- 正常语速 220 字/分钟
- 自动评估页面复杂度（要点数/数据/图表）
- 边界检查：单页 15 秒 - 5 分钟
- 支持自定义总时长自动分配

## 如何使用

这些 Skills 是为 [Claude Code](https://claude.ai/code) 设计的，需要在 Claude Code 中配置使用。

1. 将 skill 目录复制到你的 Claude Code skills 目录（`~/.claude/skills/`）
2. 或在 Claude Code 设置中配置 skill 路径
3. 使用 `/skill-name` 命令触发对应技能
