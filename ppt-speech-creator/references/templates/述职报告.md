# 述职报告PPT模板

## 适用场景
- 岗位晋升述职
- 试用期转正述职
- 年度/季度绩效述职
- 岗位竞聘演讲

## 标准结构（建议7-9页）

### 第1页：封面
- 主标题：述职报告
- 副标题：[岗位名称]岗位述职
- 个人信息：姓名 | 现任岗位 | 入职时间
- 述职日期

### 第2页：个人简介
**内容要点：**
- 基本信息（岗位、司龄、汇报线）
- 岗位职责概述
- 本次述职周期

**可选内容：**
- 教育背景（晋升/竞聘时）
- 职业履历亮点
- 相关资质证书

### 第3页：工作回顾
**内容要点：**
- 核心职责履行情况
- 关键工作清单
- 工作量/产出统计

**展示方式：**
- 职责对照表（职责→行动→结果）
- 时间轴（重要节点）
- 数据仪表盘（产出数量）

### 第4页：核心业绩
**内容要点：**
- 3-5项核心业绩
- 每项业绩的结构：背景→行动→结果→影响

**STAR法则：**
- **S**ituation：什么背景/问题
- **T**ask：你的任务/目标
- **A**ction：你采取的行动
- **R**esult：量化结果

**业绩选择原则：**
1. 与岗位职责高度相关
2. 有量化数据支撑
3. 体现个人独特贡献
4. 展示成长和能力提升

### 第5页：能力提升
**内容要点：**
- 硬技能提升（专业领域）
- 软技能成长（沟通、管理等）
- 学习成果（培训、认证）

**展示维度：**
| 能力维度 | 入职时 | 现在 | 提升方式 |
|---------|-------|-----|---------|
| 专业技能 | | | |
| 项目管理 | | | |
| 团队协作 | | | |

### 第6页：不足与反思
**内容要点：**
- 客观存在的不足（2-3点）
- 原因分析
- 已采取的改进措施

**表达技巧：**
- 真诚但不自我贬低
- 聚焦可改进的方面
- 展示自我认知能力
- 每个不足都有改进方案

### 第7页：未来规划
**内容要点：**
- 岗位目标（6-12个月）
- 能力提升计划
- 重点工作方向

**目标设定SMART原则：**
- **S**pecific：具体明确
- **M**easurable：可衡量
- **A**chievable：可实现
- **R**elevant：相关性强
- **T**ime-bound：有时限

### 第8页：总结与展望
**内容要点：**
- 一句话总结核心价值
- 对岗位/公司的承诺
- 感谢与期待

### 第9页：致谢
- 感谢聆听
- 欢迎提问
- 联系方式

## 不同类型述职侧重点

### 晋升述职
**重点突出：**
- 已具备目标岗位的能力
- 超出当前岗位的贡献
- 带领/影响他人的案例
- 战略思维和全局观

**结构侧重：**
- 业绩：40%
- 能力：30%
- 规划：20%
- 其他：10%

### 转正述职
**重点突出：**
- 快速融入和适应
- 岗位胜任度
- 文化认同
- 学习成长速度

**结构侧重：**
- 工作：50%
- 融入：25%
- 规划：15%
- 其他：10%

### 竞聘述职
**重点突出：**
- 与竞聘岗位的匹配度
- 差异化竞争优势
- 上任后的具体计划
- 对岗位的独特理解

**结构侧重：**
- 匹配度：30%
- 业绩：25%
- 规划：30%
- 优势：15%

## 时长分配建议

| 页面 | 建议时长 | 说明 |
|-----|---------|-----|
| 封面 | 10秒 | - |
| 个人简介 | 30秒 | 快速带过 |
| 工作回顾 | 1.5分钟 | 职责履行 |
| 核心业绩 | 4-5分钟 | 重点展开 |
| 能力提升 | 1分钟 | 成长展示 |
| 不足反思 | 1分钟 | 真诚简洁 |
| 未来规划 | 2分钟 | 展示思考 |
| 总结展望 | 30秒 | 收尾 |
| 致谢 | 10秒 | - |

**总计：约10-12分钟**

## 数据准备清单

述职前准备以下数据：

**业绩数据：**
- [ ] 量化指标完成率
- [ ] 项目交付数量/质量
- [ ] 效率提升数据
- [ ] 成本节约数据
- [ ] 客户/用户满意度

**过程数据：**
- [ ] 工作时长/投入度
- [ ] 协作项目数量
- [ ] 跨部门沟通频次
- [ ] 问题解决数量

**成长数据：**
- [ ] 培训参与/完成
- [ ] 新技能掌握
- [ ] 获得的认可/奖励

## 常见错误避免

1. ❌ 流水账罗列工作
   ✅ 按重要性排序，突出关键业绩

2. ❌ 只有苦劳没有功劳
   ✅ 用数据说话，量化贡献

3. ❌ 团队成果个人化
   ✅ 明确个人在团队中的独特贡献

4. ❌ 避谈不足
   ✅ 真诚反思，展示成长型思维

5. ❌ 规划空洞
   ✅ 具体、可衡量的目标和行动计划

6. ❌ 超时
   ✅ 严格控制时间，重点前置

## 评委关注点（晋升/竞聘）

**能力维度：**
1. **专业能力**：是否胜任目标岗位的专业要求
2. **管理能力**：是否有带领团队的潜质
3. **战略思维**：是否有全局观和前瞻性
4. **执行力**：是否能把想法落地
5. **影响力**：是否能影响他人、推动协作

**品质维度：**
1. **价值观**：是否与公司文化匹配
2. **成长性**：是否有持续学习的意愿和能力
3. **抗压性**：面对挑战的态度和表现
4. **责任心**：对结果的担当

**准备建议：**
- 提前了解评委构成和关注点
- 准备业绩的详细支撑材料
- 预设可能的提问和回答
